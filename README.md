# Clean It

A gamified chore-tracking app that turns tidying up into a game. Complete chores, earn coins and XP, build streaks, and unlock badges.

## Features

- **Daily progress** — track how many chores you've completed today and this week, plus your current tidy streak
- **Chores by room** — chores are grouped by room (e.g. Main Bedroom, Kitchen) with per-room completion progress, due dates (Today, Tomorrow, This week, Overdue), and XP/coin rewards
- **Leveling & XP** — earn experience points toward your next level, with progress shown on your dashboard
- **Coins & spending** — earn coins for completed chores and spend them via the Spend action
- **Weekly quests** — complete objectives like "Spotless Week" or "Kitchen Hero" to earn coins, XP, or badges (resets weekly)
- **Badges** — unlock achievements such as First Sweep, Tidy Streak, Coin Stacker, Kitchen Boss, Night Owl, and Level milestones

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- React
- TypeScript

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Available Scripts

- `npm run dev` — start the development server
- `npm run build` — build the app for production
- `npm run start` — run the production build
- `npm run lint` — lint the codebase
- `npm run typecheck` — run TypeScript type checking
